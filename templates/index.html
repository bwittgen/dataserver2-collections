Welcome to Dataserver2 Collections Manager

This application helps you manage your Kometa (formerly PMM) collections for Plex.

API Endpoints

Available API endpoints (requires API key in X-Api-Key header):

  • /api/status - Get application status
  • /api/collections - List all collection files

Example Usage

curl -H "X-Api-Key: YOUR_API_KEY" http://localhost:5000/api/status
curl -H "X-Api-Key: YOUR_API_KEY" http://localhost:5000/api/collections
